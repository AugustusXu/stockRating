# 评分体系与指标说明



综合评分 = 期权维度评分 + 现货维度评分 + 交叉共振加分

- 每只标的的评分是当天独立计算的，不累积历史分数
- 评分越高，说明该标的在越多维度上同时出现异动信号
- 理论满分 35 分：期权最高 21 分 + 现货最高 9 分 + 共振最高 5 分
- 排名变化（↑↓）对比的是前一天的排名位置，NEW 表示前一天未上榜
- 标的筛选条件：日均成交额 ≥ $10M（排除低流动性标的），排除杠杆/反向 ETF、权证等非常规品种
- 所有数据来源 Polygon API，价格均为收盘价

## 一、期权维度评分（满分 21 分）
期权数据能看到「聪明钱」的布局——机构买了什么、买多少、押注什么方向，比股价更早暴露意图

| 指标名称 | 分值 | 什么意思（生活化类比） | 怎么用 |
| --- | --- | --- | --- |
| Vol/OI 极端 | 1 / 3 / 5 | 成交量 vs 持仓量的比率。好比一家餐厅平时每天100人，突然某天涌入500人——这不正常。比率≥10倍=5分（史诗级）,≥5倍=3分,≥3倍=1分 | 数字越大说明有人在短时间内突击建仓，可能是知道了什么内幕消息 |
| P/C Ratio<br>(看跌/看涨比) | 1 / 3 | 看跌期权和看涨期权的成交量比值。小于0.3说明几乎全部人都在买涨=3分,小于0.5偏看涨=1分,大于2.0恐慌买保险=1分,大于3.0极端恐慌=3分 | P/C < 0.3 要警惕「拥挤交易」——全员看涨时一旦翻转会踩踏<br>P/C > 2.0 反而可能是底部信号（所有人都在买保险 = 恐惧到极点） |
| IV Skew 反转 | 3 | 正常情况下看跌期权比看涨期权贵（因为大家怕跌）。一旦反转（Call比Put贵），说明有人不惜溢价也要买涨。这是最难伪造的信号之一 | 出现 IV Skew 反转 + P/C < 0.3 = 非常强的看涨确认 |
| 多行权价联动 | 2 | 如果只在一个价位出现异动，可能是偶然; 但如果3个以上行权价同时放量，说明资金在「扫货」，是有计划的大规模布局 | 看联动的行权价分布——都在虚值Call说明赌大涨，都在实值说明稳健建仓 |
| Delta 加权方向 | 3 | Delta就是期权对股价变动的敏感度。把所有成交合约的Delta加权汇总，可以算出「净方向押注」。极端看涨或极端看跌=3分 | 净Delta > 0 = 整体看涨, < 0 = 整体看跌。数字越大方向越确定 |
| IV/HV 比率<br>(Natenberg核心) | 2 / 3 | 隐含波动率（期权价格推出来的）除以历史波动率（股价实际走出来的）。好比保险费/实际出险率：<br>> 2.0 = 保费是出险率的2倍，期权极贵 → 3分<br>> 1.5 = 保费明显偏高 → 2分<br>< 0.7 = 保费打折，期权便宜 → 2分<br>< 0.5 = 保费半价，罕见便宜 → 3分 | IV/HV > 1.8：不建议买入期权（太贵了），可以考虑卖出策略<br>IV/HV < 0.7：期权被低估，买入时机。可考虑跨式(Straddle)套利 |
| 末日期权效应 | 2 | 到期日≤5天的期权，Gamma（加速度）急剧放大——小的股价波动就能引起期权价格暴涨暴跌。如果此时已有其他信号加持，额外+2分 | 末日期权纯赌博成分高，但一旦方向对了收益是其他期限的数倍 |

## 二、现货维度评分（满分 9 分）
现货（股票）本身的技术面和资金面信号，用来验证期权信号是否有现货行情的支撑

| 指标名称 | 分值 | 什么意思 | 怎么用 |
| --- | --- | --- | --- |
| 高位放量 | 3 | 股价在近期高位（≥历史高点75%），同时5天平均成交量≥20天平均的1.5倍。好比一只股票已经涨了不少，但仍然有大资金在继续加仓 | 高位放量可能是「突破确认」，也可能是「出货诱多」——结合期权信号判断 |
| 强阳柱 | 1 | 当天涨幅超过5%。一根大阳线 | 单独看意义不大，需要配合放量和期权信号 |
| RS vs SPY<br>(相对强度) | 1 / 2 | 这只股票过去20天相对标普500的超额收益。跑赢>10%=2分,跑赢>5%=1分 | RS高的标的说明资金持续偏好，是市场当前的「宠儿」 |
| 连续放量≥3天 | 1 | 连续3天以上成交量超过20日均量的1.3倍。区分「脉冲」和「趋势」——一天放量可能是偶然，三天连续放量说明是持续性的资金流入 | 连续放量 + 价格上升 = 健康的趋势确认 |
| 布林带压缩 | 1 | 布林带收窄到极致，说明波动率极低——暴风雨前的宁静。一旦突破，往往是方向性大行情 | 压缩本身不指示方向。需等突破方向确认后再行动 |
| 波动率扩张 | 1 | 短期波动率正在超越长期波动率，行情开始加速 | 扩张初期是入场窗口，扩张到极致则注意回调风险 |

## 三、交叉共振加分（满分 5 分）
当期权信号和现货信号同时达标且方向一致时，额外加分。这是最高质量的信号——两个独立维度互相验证

| 共振类型 | 加分 | 触发条件 | 含义 |
| --- | --- | --- | --- |
| 🔥 真共振 (HOT) | +5 | 期权分≥3 且 现货分≥3，并且两者推断的方向一致（同看涨或同看跌） | 最高质量信号。期权市场的聪明钱和现货市场的资金流同时押注同一方向，可信度最高 |
| ⚡ 半共振 (WARM) | +3 | 期权分≥3 且 现货分≥3，但有一方方向不明确 | 信号质量次优。两个维度都有异动但方向不完全确认，需要更多信息 |
| 方向矛盾 | +1 | 期权分≥3 且 现货分≥3，但方向相反 | 可能是机构在用现货和期权做对冲。信号质量最低，不宜重仓 |

## 四、Excel 数据列含义速查
| 列名 | 数据类型 | 说明 | 判读技巧 |
| --- | --- | --- | --- |
| 排名 | 数字 | 按综合分从高到低排列 | #1 = 当天异动最强的标的 |
| 排名变化 | ↑N / ↓N / NEW | 和前一天排名的对比 | ↑10 = 比昨天上升10名，NEW = 昨天不在榜上（新出现的异动） |
| 综合分 | 数字 | = 期权分 + 现货分 + 共振加分 | ≥25 极高异动 / ≥15 值得关注 / ≥8 边缘异动 |
| 方向信号 | 标签 | 基于 P/C Ratio、IV Skew、Delta 加权方向三个指标投票得出的方向判定。<br>每个指标独立投票（看涨/看跌/弃权），综合得出方向共识。<br>⚠️ 综合分衡量的是「异动强度」而非方向——高分 + 看跌完全可能出现 | 🟢 强看涨 / 看涨 / 偏看涨：多数指标指向上涨<br>🔴 强看跌 / 看跌 / 偏看跌：多数指标指向下跌<br>⚡ 多空博弈：看涨和看跌信号势均力敌（如 APPN：P/C 看跌但 IV Skew 看涨）<br>⚪ 中性：无明确方向信号<br>带「有分歧」表示投票不一致但有一方略占优 |
| P/C Ratio | 小数 | 看跌期权量 ÷ 看涨期权量 | < 0.3 极端看涨 / 0.5-0.8 中性 / > 2.0 极端看跌 |
| ATM IV% | 百分比 | 平值期权的隐含波动率 | 越高说明市场预期越大的波动（但不指示方向） |
| IV/HV | 倍数 | 隐含波动率（期权价格推出来的）÷ 历史波动率（股价实际走出来的） | > 1.5 期权贵（卖方有利）/ < 0.7 期权便宜（买方有利） |
| Delta方向 | 数字 | 全市场Delta净敞口 | 正数 = 整体看涨 / 负数 = 整体看跌 / 绝对值越大方向越确定 |
| RS% | 百分比 | 20天相对SPY的超额收益 | > 0 跑赢大盘 / < 0 跑输大盘 |
| γ Wall | 价格 | Gamma Wall（做市商最大Gamma暴露的行权价） | 股价接近此处时会被「吸住」形成支撑/阻力 |
| GEX净值 | 数字 | 全市场净Gamma Exposure（Call GEX + Put GEX 汇总） | 正数=做市商逆势对冲（压制波动），负数=做市商顺势对冲（放大波动）<br>绝对值越大，做市商的影响力越强 |
| GEX环境 | 标签 | 当前股价所处的Gamma区域 | 正Gamma：股价倾向被「吸住」，波动收窄，γ Wall是真实阻力<br>负Gamma：做市商追涨杀跌，波动放大，突破后单边行情风险 |
| Zero Gamma | 价格 | GEX从正翻转为负值的行权价（Gamma翻转警戒线） | 正Gamma环境下：跌破Zero Gamma → 进入负Gamma区 → 波动爆炸<br>负Gamma环境下：涨过Zero Gamma → 回到正Gamma区 → 波动收窄 |
| IV Skew | 数字 | Call IV - Put IV | 正数 = 看涨情绪反转（罕见且强烈） |
| 布林压缩 | 标签 | 布林带宽度是否收窄到极致（暴风雨前的宁静） | 极度压缩(<2%)：突破迫在眉睫，方向确认后可重仓<br>压缩中(<3%)：波动率低位蓄力，关注突破方向 |
| 波动率状态 | 标签 | 短期HV(10d) vs 长期HV(50d)的比率 | 扩张：短期波动＞长期，行情加速中<br>压缩：短期波动＜长期，蓄力等待突破<br>正常：波动率处于均衡区 |
| 共振 | 标签 | 期现交叉验证结果 | 🔥真共振 > ⚡半共振 > 空白 |
| DTE | 天数 | 最热门到期日距今天数 | ≤1天=末日期权 / ≤14天=近期事件 / >14天=中期布局 |

## 五、正股看涨筛选（Sheet 3）— 通过期权聪明钱信号预测正股走势
核心思路：我们不买期权，而是读懂「期权市场的聪明钱在告诉我们正股要往哪走」。筛选出多维度信号同时看涨的标的，直接买入正股。

### 5.1 硬性筛选条件（必须全部满足才进入评分）
| 筛选条件 | 为什么？ |
| --- | --- |
| 魔眼排名 ≤ 100 | 只分析异动信号最强的头部标的，排名靠后的标的信号质量不足以支撑正股方向判断 |
| 收盘价 > γ Wall | 股价已经突破做市商 Gamma 暴露最大的行权价（γ Wall）。突破 γ Wall 意味着做市商的对冲行为从「阻力」变成「支撑」——在正 Gamma 环境下，股价越涨做市商被迫买入越多正股对冲，形成机制性买盘 |
| 突破幅度 ≤ 15% | 太远说明行情已经走了一大段，追高风险增加。0~3% 是最佳入场窗口，8~15% 趋势已确认但空间变小 |
| 正 Gamma 环境 | 做市商在正 Gamma 区域会逆势对冲（涨了卖、跌了买），相当于给市场装了「减震器」。只有在正 Gamma 下，突破 γ Wall 才会产生持续的机制性买盘支撑 |
| Delta 方向 > 0 | Delta 加权方向代表全市场期权的净方向押注。> 0 说明聪明钱通过期权整体押注正股上涨 |

### 5.2 看涨评分 — 10 个维度打分规则
通过硬性筛选后的标的，从10个维度打分。分数越高，多个独立信号同时指向看涨的确定性越强。

| 维度 | 分值 | 评分规则 | 为什么能预测正股涨 |
| --- | --- | --- | --- |
| ① γ Wall 突破位置 | 1 / 3 / 4 | 刚突破 0~3% = 4分（最佳窗口）<br>突破 3~8% = 3分（趋势确认）<br>突破 8~15% = 1分（注意追高） | 突破 γ Wall 后做市商被迫买入正股对冲（Delta Hedging），形成持续的机制性买盘。越早入场，吃到的「对冲买盘推力」越多 |
| ② Delta 方向强度 | 1 / 2 / 3 / 4 | 净 Delta ≥ 5000 = 4分（极强）<br>≥ 2000 = 3分（强）<br>≥ 500 = 2分（中）<br>> 0 = 1分（轻度） | Delta 是聪明钱方向的直接投票。净 Delta 越大，说明大资金通过期权押注正股上涨的力度越重，方向确定性越高 |
| ③ P/C Ratio<br>(看跌/看涨比) | 1 / 2 / 3 / 4 | P/C < 0.2 = 4分（极端看涨）<br>< 0.3 = 3分（强看涨）<br>< 0.5 = 2分（偏看涨）<br>< 0.8 = 1分（中性偏看涨） | 几乎全部人都在买看涨期权，极度一致的方向预期。但 < 0.1 也要警惕拥挤交易——一旦翻转会踩踏 |
| ④ IV Skew 反转 | 2 / 3 / 4 | IV Skew > 1.0 = 4分（极端反转）<br>> 0.1 = 3分（显著反转）<br>> 0 = 2分（轻微反转） | 正常情况 Put 比 Call 贵（怕跌）。一旦反转（Call 更贵），说明有人不惜溢价也要买涨——这是最难伪造的看涨信号，因为溢价买入 Call 的成本很高 |
| ⑤ 期现共振 | 2 / 4 | 🔥 真共振 = 4分<br>⚡ 半共振 = 2分 | 期权信号和现货信号同时指向同一方向 = 两个独立维度互相验证。真共振是全系统最高质量的信号 |
| ⑥ RS% 相对强度 | 1 / 2 / 3 | RS ≥ 20% = 3分（极度强势）<br>≥ 10% = 2分（显著跑赢）<br>≥ 0 = 1分（跑赢大盘） | 过去20天持续跑赢标普500，说明资金在「用脚投票」持续买入，不是一日游的脉冲异动 |
| ⑦ IV/HV 波动定价 | 1 / 2 | IV/HV > 2.0 = 2分<br>> 1.5 = 1分 | 关键认知转变：买正股时 IV/HV 高 ≠ 坏事。方向已经通过前面的筛选确认为看涨，此时高 IV/HV = 市场在为即将到来的大波动定价，且方向已确认向上 |
| ⑧ 资金持续性 | 1 / 2 | 连续放量 ≥ 3天 = 2分<br>放量比 ≥ 2.0x = 1分 | 区分「脉冲」和「趋势」。一天放量可能是偶然，三天连续放量说明持续有新资金流入，不是一锤子买卖 |
| ⑨ 波动率状态 | 1+1 | 波动率扩张 = 1分<br>布林带压缩 = 1分 | 扩张 = 趋势正在加速。布林带压缩 = 暴风雨前的宁静，突破后行情可能更猛烈 |
| ⑩ 机构布局期限<br>(DTE) | 1 / 2 | DTE > 30天 = 2分（战略建仓）<br>DTE > 14天 = 1分（中线布局） | 聪明钱买期权选的到期日暴露了它的意图。DTE > 30天说明不是赌一把的短线，而是中长线战略建仓——这种资金更「粘」，不会轻易跑 |

**⚠️ 扣分项：当日涨幅 > 10% 扣 2 分，> 5% 扣 1 分（追高风险惩罚）**

### 5.3 关键概念解释
| 概念 | 简称 | 含义 | 实战用法 |
| --- | --- | --- | --- |
| 看涨分 | bull_score | 10个维度的加权总分，代表「期权聪明钱看涨正股」的确定性有多高。不是涨幅预测，而是「方向确定性」的量化指标。理论最高约 30+ 分 | ≥ 20分 = 多维信号高度一致，可考虑较大仓位<br>15-19分 = 信号较强但个别维度未确认，中等仓位<br>10-14分 = 基础信号通过，轻仓试探 |
| 安全边际 | safety_margin | 当前股价到 Zero Gamma 的距离百分比。<br>计算公式：(收盘价 - Zero Gamma) ÷ 收盘价 × 100%<br><br>例如：股价 $100，Zero Gamma = $90<br>→ 安全边际 = (100-90)/100 = 10%<br><br>意思是「股价要跌 10% 才会触碰到危险区」 | > 15% = 抗跌能力强，即使短期回调也不太容易跌进危险区<br>5~15% = 尚可，但要密切观察<br>< 5% = 非常接近 Zero Gamma，一个坏消息可能直接跌穿<br><br>安全边际越大，持仓越安心 |
| 突破% | excess | 股价超过 γ Wall（Gamma Wall）的百分比。<br>计算公式：(收盘价 - γ Wall) ÷ γ Wall × 100%<br><br>例如：股价 $103，γ Wall = $100<br>→ 突破 +3% | 0~3% = 最佳入场窗口（刚刚突破）<br>3~8% = 趋势已确认，仍然安全<br>8~15% = 走了一段了，追高需谨慎<br>> 15% = 不纳入筛选（太远了） |
| 梯队 | tier | 按看涨评分分级：<br>🥇 一梯队 ≥ 20分：多维信号高度一致<br>🥈 二梯队 15-19分：信号较强<br>🥉 三梯队 10-14分：基础信号 | 一梯队优先关注<br>三个梯队之间是风险递增、确定性递减的关系 |
| Zero Gamma<br>(Gamma 翻转线) | ZG | GEX（Gamma Exposure）从正值翻转为负值的行权价。<br><br>正 Gamma 区 = 做市商逆势对冲（涨了卖、跌了买）→ 压制波动、形成支撑<br>负 Gamma 区 = 做市商顺势对冲（涨了买、跌了卖）→ 放大波动、加速暴跌<br><br>Zero Gamma 就是这两个区域的分界线 | 🛡️ Zero Gamma 是正股操作的核心止损位<br><br>跌破 Zero Gamma = 从「有人托底」变成「有人踩踏」<br>→ 波动突然放大、下跌加速<br>→ 必须止损出场或大幅减仓 |

### 5.4 正股操作建议速查
- 仓位管理：一梯队（≥20分）可较大仓位 → 二梯队（15-19分）中等仓位 → 三梯队（10-14分）轻仓试探
- 止损位置：统一以 Zero Gamma 为关键止损线。跌破 = 做市商从「压制波动」变成「追涨杀跌」，下跌将加速
- 入场时机：突破 γ Wall 越近越好（0~3% 最佳），当日已大涨 >5% 的建议等回调
- 关键认知：IV/HV 高在此语境下不再是坏事 — 方向已确认看涨，高 IV/HV = 市场预期大波动且方向向上
- 信号验证：看涨分高只代表方向确定性高，不等于一定会涨。建议结合基本面和大盘环境综合判断
